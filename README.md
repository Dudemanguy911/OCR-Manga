# OCR-Manga
A very simplistic manga reader with integrated OCR.

Dependencies
------------
- Python 3
- [pyocr] (https://github.com/jflesch/pyocr "pyocr")
  - only tested with tesseract
- [myougiden](https://github.com/leoboiko/myougiden "myougiden")
